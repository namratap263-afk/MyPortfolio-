Featured Project • Hotel Booking Data Analysis & Dashboard

Hotel Booking Data Analysis & Dashboard

Problem Statement

Analyze hotel booking patterns to identify trends in occupancy, cancellations, booking channels, and customer behavior that could support operational and revenue decisions.

Dataset

Used a real hotel booking dataset with booking dates, customer segments, stay durations, room types, and reservation status.

Excel Cleaning

Cleaned and structured the dataset in Excel by handling missing values, standardizing fields, removing inconsistencies, and preparing it for analysis.

KPIs

  • Total bookings and cancellations
  • Average lead time and stay duration
  • Booking channel performance
  • Room type and market segment trends

Dashboard

Built a clear dashboard layout highlighting booking flow, demand trends, and performance insights in a visual, recruiter-friendly format.

Business Insights

Identified booking patterns that could help improve planning, improve occupancy consistency, and better understand customer preferences.

Results

Delivered a practical analysis workflow that converts raw reservation data into understandable metrics and decision-ready visuals.

Tools Used

  • Excel
  • Pivot Summary Table
  • Data Cleaning
  • Dashboard Design
  • Power Query
